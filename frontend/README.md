# Wedding Bingo Frontend

This is the frontend part of the Wedding Bingo application, built with React and Vite.

## Setup

1. Install dependencies:
   ```
   npm install
   ```

2. Run the development server:
   ```
   npm run dev
   ```

The frontend will run on `http://localhost:5173` by default.

## API

The frontend communicates with the backend API running on `http://localhost:5000`.

Make sure the backend is running before using the frontend features like gallery and contact form.
