# Wedding Bingo

A full-stack wedding photography website with image gallery, contact form, and email notifications.

## Project Structure

- `backend/`: Node.js/Express server with MongoDB
- `frontend/`: React/Vite frontend application

## Quick Start

1. Install all dependencies:
   ```
   npm run install:all
   ```

2. Set up backend environment (see backend/README.md)

3. Run both frontend and backend:
   ```
   npm run dev
   ```

This will start the backend on port 5000 and frontend on port 5173.

## Individual Runs

- Backend only: `npm run dev:backend`
- Frontend only: `npm run dev:frontend`

## Features

- Responsive wedding photography website
- Image gallery with like/click tracking
- Contact form with email notifications
- Admin email notifications for inquiries
